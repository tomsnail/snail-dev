动态监控 精确帮扶 贵阳编织数字化禁毒网

时间：2017-03-30 09:48:30     作者：商 旸     来源：人民日报

贵阳毒患，曾经异常顽固。

    针对禁毒工作的“痛点”，贵阳市创造性地提出变“部门首责”为“党委首责”，破解“落实不力”，带动多部门联动共同担当；利用“大数据”织就一张精密、详实的涉毒人员信息网络，动态监控，精确帮扶，实现一对一帮扶戒毒。

    通过不断努力，贵阳市禁毒人民战争取得了多项突破，戒治水平不断提升：2015年吸毒人员动态管控工作综合考评排名贵州省第一，全市符合条件的出所吸毒人员必接率达99%以上，戒断三年未复吸率达38.2%，社区戒毒社区康复执行率达98.72%，多项指标高于全国、全省平均水平。

    力推党政主导新模式

    禁毒人民战争开展以来，贵阳市着力推进党政主导新模式，实现市、县、乡三级党委书记任同级禁毒委第一主任，市（县、乡）长任主任的党政禁毒工作格局；研究制定了《贵阳市禁毒人民战争考核奖惩办法》，建立了以青少年吸毒人员新增率、收戒管控率、戒断巩固率、市民对禁毒工作满意度为核心指标的考评体系。

    “党政主导是首要关键，特别是市委常委会每两个月调度，政府各职能部门和区（市、县）切实履行工作职责，构建了‘条专块统、条块结合’的工作体系，改变公安机关单打独斗的局面，为禁毒工作全面推进奠定了基础。”贵州省委常委、贵阳市委书记陈刚说。

    党委来牵头，书记亲自抓，实现了全市上下全面动员。民政部门牵头建设以“阳光驿站”为主题的社会化戒毒康复站；人社部门在各新型戒毒所设置了培训基地，确保每一位戒毒人员“经过一次培训、掌握一门技能、拿到一本证书”；体育部门派驻教员，开展身体强健教育；教育部门设置“文化教育学校”，开展扫盲、文化提升和国学教育。贵阳全市还建立禁毒志愿者协会92个，吸纳成员5100多人。

    乌当区依托“阳光驿站”成立的社区公益堡，引进专业社工组织，发动爱心企业、爱心人士和志愿者等社会力量共同打造的公益平台，通过政府购买服务、以奖代补等方式，链接各类社会资源，对戒毒康复人员开展宣传教育、走访谈心等各类融入社会的主题活动。

    贵阳市建设了“科学戒管”工作实体。根据吸毒人员戒毒、康复、回归社会的不同阶段，投入14.82亿元开工建设7个新型戒毒所，建成34个社会化戒毒康复站、30个“阳光工程”就业安置点、11个“特殊病区”， 招录了1152名禁毒专职工作人员，形成了工作实体间环环相扣的“无缝衔接”机制。

    “数据驱动”提升禁毒效率

    以实施“社会和云”工程为契机，贵阳建成了“数据禁毒系统”并投入使用，力求实现信息、动态、管控、戒治、帮扶、考核六联动。

    数据从社区、乡镇、社会化戒毒康复站、就业安置机构、维持治疗机构、特殊病区、戒毒场所、公安办案部门七大数据源，汇聚到系统中。从个人经历、吸毒史、身心状况、家庭情况，到社戒社康执行情况、违法记录、强制隔离戒毒情况、美沙酮治疗情况、就业安置情况，数据收录事无巨细。

    这些数据经过系统的自组织化分析，生成了管控积分及分类分级结果。积分不仅是社区、乡镇和公安机关的管控依据，也能精确指导社区戒毒康复工作。

    数据戒毒，精确帮扶，贵阳以“数据驱动”为必要手段，通过精确到人、精确到事的指令推送实现了精确化管控，大大提升了禁毒工作的效率。
